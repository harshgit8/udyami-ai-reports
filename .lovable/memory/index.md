# Project Memory

## Core
Brand is exclusively 'Udyami' (no sub-branding). AI acts as a high-level MSME manufacturing consultant (no generic AI phrasing).
AI Guardrails: Strictly locked to polymer/MSME manufacturing. Defends against prompt injection and rejects unrealistic inputs.
Design: Minimalist black-and-white (white bg, black text). Bauhaus + Apple/macOS style. Soft shadows, rounded cards, clean typography.
Stack: React, TypeScript, Tailwind, Supabase (with Edge Functions), Gemini, Recharts.
