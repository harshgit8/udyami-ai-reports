# Udyami AI

Advanced Industrial Document Management & AI Assistant.

## Features

- **Quotations**: Generate professional quotes with cost breakdowns and win probability analysis.
- **Invoices**: Manage tax invoices with payment tracking and risk assessment.
- **Quality Inspection**: Analyze batch reports, defect rates, and compliance.
- **Production Schedule**: AI-driven scheduling and machine allocation.
- **R&D Formulations**: Material property predictions and compliance checks.
- **AI Chat Assistant**: Industrial manufacturing operations expert.

## Tech Stack

- React & TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Supabase Edge Functions
- Google Gemini AI

## Getting Started

### Prerequisites

- Node.js & npm

### Installation

1. Clone the repository
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm run dev
   ```
